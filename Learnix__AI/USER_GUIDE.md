# Learnix AI - User Guide

Welcome to Learnix AI, your smart study assistant! This guide will help you make the most of all the features.

## 📱 Getting Started

### 1. Sign Up

1. Open Learnix AI in your browser
2. Click **"Sign Up"** on the login page
3. Fill in your details:
   - Full Name
   - Email Address
   - Password (minimum 6 characters)
   - Confirm Password
4. Click **"Sign Up"**
5. You'll be redirected to login page

### 2. Login

1. Enter your email and password
2. Click **"Login"**
3. You'll be taken to your dashboard

## 🏠 Dashboard

The dashboard shows your learning progress at a glance:

- **Total Notes Uploaded**: Number of study materials you've uploaded
- **Quizzes Completed**: Total quizzes you've taken
- **Average Score**: Your overall quiz performance

### Quick Tips on Dashboard

- Review your statistics regularly to track progress
- Follow the "Get Started" guide for new features
- Check the Pro Tip about setting up your Groq API key

## 🔑 Setting Up Groq API Key

To use AI features, you need a Groq API key:

1. Visit [console.groq.com](https://console.groq.com)
2. Create a free account
3. Navigate to API Keys section
4. Generate a new API key
5. Copy the key
6. In Learnix AI, go to **"AI Summary"** page
7. Click **"Set API Key"**
8. Paste your key and click **"Save"**

**Note**: Your API key is stored securely in your browser and never shared.

## 📤 Upload Notes

### Supported File Types

- PDF files (.pdf)
- Text files (.txt)

### How to Upload

1. Go to **"Upload Notes"** from the sidebar
2. Click **"Choose File"** or drag and drop
3. Select your file (max 10MB)
4. Wait for upload to complete
5. Your file appears in "Your Uploaded Files"

### Managing Files

- **View Files**: See all your uploaded materials
- **Delete Files**: Click the trash icon to remove a file
- **File Information**: See upload date and file size

## ✨ AI Summary Generator

Generate summaries and key points from your notes using AI.

### Generate Summary

1. Go to **"AI Summary"** from sidebar
2. Select a file from dropdown
3. Click **"Generate Summary"**
4. Wait for AI to process (10-30 seconds)
5. Read and save the summary

### Generate Key Points

1. Select a file
2. Click **"Generate Key Points"**
3. AI extracts the most important points
4. Use these for quick revision

### Tips

- Upload clear, well-structured notes for best results
- Use summaries before exams for quick review
- Combine summary with key points for comprehensive understanding

## ❓ Question Generator

Generate exam-style questions with answers.

### 2-Mark Questions

1. Go to **"Question Generator"**
2. Select a file
3. Click **"Generate 2-Mark Questions"**
4. Get 10 short questions with answers
5. Practice these for concept clarity

### 16-Mark Questions

1. Select a file
2. Click **"Generate 16-Mark Questions"**
3. Get 5 detailed questions with comprehensive answers
4. Use for in-depth exam preparation

### Study Tips

- Answer questions before looking at AI answers
- Compare your answers with AI answers
- Identify knowledge gaps and study those areas

## 🎯 Quiz Generator

Test your knowledge with AI-generated multiple choice quizzes.

### Taking a Quiz

1. Go to **"Quiz Generator"**
2. Select a file
3. Click **"Start Quiz (10 Questions)"**
4. Wait for quiz generation
5. Answer each question by selecting an option
6. Use **"Next"** to move forward
7. Use **"Previous"** to go back
8. Click **"Submit"** on the last question

### Quiz Interface

- **Progress Bar**: Shows how far you've progressed
- **Answered Counter**: Tracks completed questions
- **Timer**: Helps manage your time
- **Options**: Select A, B, C, or D

### After Quiz

- **Score Display**: See your percentage and score
- **Review Answers**: See which questions you got right/wrong
- **Correct Answers**: Highlighted in green
- **Wrong Answers**: Your selections in red
- **Take Another**: Start a new quiz anytime

### Performance Tracking

- All quiz scores are saved
- View history on dashboard
- Track improvement over time

## 💬 AI Chatbot

Your personal study assistant for questions and explanations.

### How to Use

1. Go to **"AI Chatbot"**
2. Type your question in the input box
3. Press **Enter** or click **"Send"**
4. AI responds with simple explanation
5. Continue the conversation as needed

### Question Examples

- "Explain photosynthesis in simple terms"
- "What is the difference between RAM and ROM?"
- "How does a for loop work in Python?"
- "Explain Newton's laws of motion"

### Tips for Best Results

- Be specific in your questions
- Ask one concept at a time
- Ask follow-up questions for clarity
- Use for doubt clearing and concept understanding

### Chat Features

- **Conversation History**: See all messages
- **Time Stamps**: Track when messages were sent
- **Scroll to Latest**: Automatic scroll to newest message
- **Real-time Responses**: Get instant AI answers

## 📅 Study Planner

Create a personalized study schedule for your exams.

### Creating a Study Plan

1. Go to **"Study Planner"**
2. Select your exam date
3. Add subjects:
   - Type subject name
   - Click **"Add Subject"** for more
   - Remove with **X** button
4. Click **"Generate Study Plan"**
5. AI creates a day-by-day schedule

### Using Your Study Plan

- **Daily Schedule**: What to study each day
- **Time Allocation**: Suggested study duration
- **Topic Coverage**: Specific topics to cover
- **Breaks**: Recommended rest periods
- **Revision Days**: Built-in review sessions

### Study Plan Tips

- Follow the schedule consistently
- Mark completed topics
- Adjust if you fall behind
- Take recommended breaks
- Review regularly

## 🐛 Troubleshooting

### AI Features Not Working

**Problem**: "Please set your Groq API key"
**Solution**: Go to AI Summary page and set your API key

**Problem**: "Failed to generate"
**Solution**: 
- Check internet connection
- Verify API key is correct
- Try again after a few seconds

### File Upload Issues

**Problem**: "Failed to upload file"
**Solution**:
- Check file size (max 10MB)
- Ensure file type is PDF or TXT
- Try different file

### Login Issues

**Problem**: "Invalid email or password"
**Solution**:
- Check email spelling
- Verify password is correct
- Try password reset (if available)

### Quiz Issues

**Problem**: Quiz not loading
**Solution**:
- Refresh page
- Check file has content
- Try different file
